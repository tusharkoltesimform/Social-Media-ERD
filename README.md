# Social-Media-ERD

This repository contains an **Entity Relationship (ER) Diagram** for a social media platform that models users, content, interactions, communities, events, and advertising.

##  Key Entities

* **User** – Platform users and profile data
* **Posts** – User-generated content
* **Comments** – Post discussions with threaded replies
* **Likes** – Likes on posts or comments
* **Follows** – User-to-user following
* **Groups & Group Members** – Community feature (many-to-many)
* **Events & RSVP** – Event creation and participation
* **Messages** – Private one-to-one communication
* **Advertisements & Stats** – Monetization and analytics
